# XP Gains - RPG Fitness App

XP Gains is an Android app that turns your real-world workouts into RPG character progression. As you exercise, your 2D hero gains stats, levels up, and earns gear to embark on missions.

## Features
- Create your own 2D RPG hero
- Track workouts and convert them into XP and stats
- Level up your hero
- Equip gear and complete missions (future version)
- Save and load progress
